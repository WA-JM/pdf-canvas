WA PDF Reader — help

Scroll
Read up/down within a document — each column scrolls independently
Shift+Scroll
Move sideways between documents
Ctrl/⌘+Scroll
Zoom toward your cursor
H / M
Highlighter / red marker — drag to draw
S Snip
Drag a rectangle → that region is copied to your clipboard
E Erase / V Read
Click a mark to remove it / return to scrolling
Select text
In Read mode, drag to select then Ctrl+C to copy (e.g. quotes from filings)
Ctrl+Z
Undo last mark or erase — Ctrl+Shift+Z to redo
Tab / Shift+Tab
Step to the next / previous document and center it (the focused doc gets a gold border). Or click a title bar / sidebar entry.
Ctrl+← / →
Also jumps between documents
↑ ↓ / PgUp PgDn
Scroll within the selected document (also Space, Home/End)
Ctrl+↑ / ↓
Jump to the previous / next page of the selected document
Ctrl+F
Find in the active document — Ctrl+Shift+F searches all documents. Enter / Shift+Enter step through hits.
Ctrl+P
Print the selected document
Ctrl+Enter
Full-screen / theater reading mode (Esc to exit)
Ctrl+1 / 2
Zoom in / out (also + / −, or Ctrl+scroll)
.pdfx: “Save .pdfx” merges all documents into one normal PDF (opens anywhere); re-open it here to split back into columns. Annotations are not yet burned into the saved file — use Snip to capture marked-up regions.
